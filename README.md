# Graphical_Models_2021

This is the network architecture for our paper published in Graphical Models (CCF-B).

### Requirements:
tensorflow-gpu

numpy

scipy

pillow

TensorFlow 0.11+

python 3.6+

### Usage:
$\bullet$ train the model:
```
python main.py --mode train
```
$\bullet$ test the model
```
python main.py --mode test
```
